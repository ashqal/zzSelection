how to use it?

  • 1、head import jquery-1.8.2.min.js
  • 2、head import jquery.timers.min.js
  • 3、head import zzSelection/js/zzSelection-1.0.js
  • 4、head import zzSelection/css/zzSelection.css
  • 5、set the select class = "zzSelection"
  • 6、enjoy it as native html select!

note:the zzSelection is float left by default!
